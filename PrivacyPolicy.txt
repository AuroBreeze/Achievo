Achievo 隐私政策（v1.0）

更新日期：2025-10-02
生效日期：2025-10-02

本隐私政策适用于 Achievo（“本应用”）。我们重视用户隐私，承诺在合法、正当、必要的前提下处理您的信息。

一、我们收集或处理的信息
1. 本地数据处理（默认）
   - 您在设置中选择的 Git 仓库路径及仓库内代码变更（增删行统计、提交历史元数据）在本地设备上处理和存储。
   - 本应用默认不将您的代码、提交内容或统计数据上传至任何我们的服务器（我们没有自建云端）。

2. 第三方 AI 服务（可选）
   - 如您在设置中启用 AI 总结功能并配置 OpenAI、DeepSeek 或自定义 API Key，本应用会将为生成总结所需的最小必要上下文（如变更摘要、统计信息及必要片段）发送至您选择的第三方服务商接口。
   - 发送的具体内容取决于您当次操作及本地生成的摘要。您可在不启用 AI 功能的前提下完全避免对外发送任何代码相关内容。

3. 配置与偏好
   - 本地保存的设置项包括：已选择的仓库路径、AI 服务端点与 API Key（加密/安全存储策略见“八、数据安全”）、界面偏好、轮询间隔等。

二、我们如何使用信息
1. 提供核心功能：统计代码变更、生成可视化图表、计算评分与趋势。
2. 可选 AI 功能：在您明确开启的情况下，将必要数据提交至第三方 AI 服务以生成总结与分析。
3. 体验优化：本地保存设置与偏好以改善使用体验。

三、信息共享与对外提供
1. 我们不出售您的个人信息。
2. 我们不对外共享您的本地代码或统计数据，除非：
   - 您主动启用并使用第三方 AI 服务；
   - 法律法规或主管机关要求（在依法合规、最小化范围内）。
3. 第三方 AI 服务会依据其各自的隐私政策处理您提交的数据，请在启用前仔细阅读其条款（例如 OpenAI、DeepSeek 等）。

四、您的选择与控制
1. 您可随时在设置中关闭 AI 功能，关闭后本应用不再向第三方服务发送任何数据。
2. 您可随时更换或删除本地配置（包括 API Key）。
3. 您可以通过删除本地应用数据的方式移除已存储的配置与缓存（见“七、数据保存与删除”）。

五、儿童隐私
本应用面向开发者与普通成年用户，不针对未成年人设计或推广。若您为未成年人，请在监护人同意与指导下使用。

六、国际传输
当您启用第三方 AI 服务时，数据可能根据该服务商的服务器部署位置在境内外传输与处理。请根据您的合规要求自行选择服务商与数据最小化配置。

七、数据保存与删除
1. 本应用的运行数据与配置默认保存在本地设备。
2. 您可通过以下方式删除：
   - 在应用内清除/重置设置；
   - 卸载应用并同时删除本地数据目录（路径因系统而异）；
   - 手动删除应用的配置文件/缓存文件（如存在）。

八、数据安全
1. 应用层安全：
   - 采用 Electron 安全最佳实践（如禁用 nodeIntegration、启用 contextIsolation、通过 preload 暴露受限 API）。
2. 凭证安全：
   - API Key 仅存储在本地，不会上传至我们的服务器；
   - 我们建议您使用系统级安全存储（如操作系统密码库）或磁盘加密，避免明文泄露；
   - 请妥善保管 API Key，并仅在必要时启用 AI 功能。
3. 传输安全：
   - 与第三方 AI 服务之间的通信应使用 HTTPS/TLS。

九、第三方服务与链接
1. 当您选择接入第三方 AI 服务时，其服务条款与隐私政策将适用该部分数据处理。
2. 应用可能包含到第三方网站或资源的链接，这些网站有独立的隐私政策与安全措施，我们不对其内容或行为负责。

十、政策更新
我们可能因功能变更、法律法规或业务调整更新本政策。重大变更将通过更新版本随附的文档或项目主页进行提示。继续使用即表示您接受更新后的政策。

十二、适用法律与争议解决
本政策的订立、履行与解释，原则上适用您所在地的强制性法律规定。因本政策引发的争议，可协商解决；协商不成的，提交有管辖权的法院诉讼解决。

附：最小化与离线优先的承诺
- 未启用 AI 功能时，代码与统计仅在本地处理与保存；
- 启用 AI 功能时，仅在必要范围内发送数据用于生成总结，且不作额外用途；
- 您可随时撤回授权并删除本地数据。
